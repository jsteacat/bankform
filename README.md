# Тестовое задание
#### Технологии:
* **SCSS**
* **Gulp**
* **jQuery**

#### Установка зависимостей и запуск
    npm i && npm i -g gulp && gulp start


#### Время выполнения
* **Нарезка и экспорт макета: _10 минут_**
* **Организация структуры проекта: _30 минут_**
* **Вёрстка: _20 минут_**
* **Стилизация: _2 часа 40 минут_**
* **Валидация: _1 час 20 минут_**
* *Всего: 5 часов*
